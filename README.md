# 基于 DataEase 的个人健身数据可视化看板
OpenRank杯初赛作品 - 数据科学大二新手项目

## 项目简介
本项目使用开源BI工具DataEase，将个人健身数据（运动类型、时长、卡路里消耗等）转化为可视化看板，直观展示运动习惯与效果，同时为DataEase提供个人健康场景的应用案例。

## 环境准备
1. 安装 DataEase：参考官方文档 https://dataease.io/docs/installation/
2. 辅助工具：Excel 2019及以上版本
